# 🧸 Mexico Toy Sales & Inventory Analysis using Metabase

🚀 I explored **Metabase with Supabase**, spinning everything up using **Docker**.This project analyzes the **Maven Toys dataset** to uncover insights on sales performance and inventory trends using interactive BI dashboards.
---

🔗 [Download Full PDF](https://github.com/MAHFUZATUL-BUSHRA/Metabase_Maven-Toys/blob/main/Metabase%20Project/Mexico%20Toy%20Sales%20%26%20Inventory%20Analysis%20using%20Metabase.pdf.pdf)
---

## 🧩 Tech Stack
- 🐳 Docker  
- 📊 Metabase  
- 🗄️ Supabase (PostgreSQL)  
- 📈 Data Visualization & BI Dashboards  

---

## 📌 Project Objectives
- Analyze toy sales performance across stores and products  
- Monitor inventory levels and stock distribution  
- Identify top-performing products and store locations  
- Build interactive dashboards for business insights

## 📊 Key Insights
- Identified best-selling toy categories and products  
- Compared store-wise revenue and inventory distribution  
- Detected potential stock shortages and overstock patterns  
- Created interactive dashboards for business monitoring  

---

## 🧠 What I Learned
- Setting up an end-to-end BI stack using Docker  
- Connecting Supabase PostgreSQL with Metabase  
- Creating insightful visualizations from relational data  
- Designing dashboards for retail sales & inventory analytics  

---

## ▶️ How to Run the Project
 
1. Run Docker containers for Supabase & Metabase  
2. Load the Maven Toys dataset into Supabase  
3. Connect Metabase to the database  
4. Explore dashboards and visualizations
